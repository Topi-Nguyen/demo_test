1. Git status
            - trạng thái hiện tại của respo đang đứng 
        2. Git branch "name"
            - tạo nhánh mới từ nhanh hiện hành
        3. git checkout "name"
            - chuyển trạng thái qua 1 nhánh đã tạo 
        4. git commit -m "abc"
            - tạo 1 cam kết khi chứng tỏ có 1 file hoặc nhiều file thay đổi 
        5. git push origin "branch"
            - đẩy những thay đổi mới nhất lên origin 
        6. git add "name"
